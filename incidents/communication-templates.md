# Incident Communication Templates

## Initial Internal Update

```text
Status: Investigating
Impact: <who or what is affected>
Start time: <known or estimated>
Current symptoms: <observable symptoms>
Evidence so far: <alerts, logs, metrics, traces, user reports>
Current hypothesis: <working theory or unknown>
Next update: <time>
Owner: <role or team>
```

## Escalation Handoff

```text
Escalation reason: <why escalation is needed>
Impact summary: <business or operational impact>
Timeline: <key timestamps>
Evidence collected: <links or sanitized excerpts>
Actions attempted: <read-only checks and approved remediation>
Current blocker: <decision, access, dependency, unknown root cause>
Requested help: <specific ask>
```

## Recovery Update

```text
Status: Mitigated or resolved
Customer impact: <current state>
Recovery evidence: <metrics, logs, synthetic checks, user confirmation>
Remaining risk: <known residual risk>
Follow-up: RCA and prevention actions
```
