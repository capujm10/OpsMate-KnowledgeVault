# Anti-Patterns to Avoid

Avoid turning the vault into:

- A generic diary
- A random Markdown dump
- A chatbot memory database
- A vector database project
- A heavy RAG platform
- A background indexing daemon
- A cloud-synced secret risk
- A collection of unsanitized production logs
- A folder of screenshots without operational interpretation
- A list of commands without context or validation
- A set of remediation recipes with no safety boundary

## Content Anti-Patterns

- Vague root causes such as "network issue"
- Blame-based RCA language
- Missing validation steps
- Missing impact description
- Missing telemetry interpretation
- Copying sensitive data for realism
- Adding too many tags that no one will search
- Writing long essays that are hard to maintain
